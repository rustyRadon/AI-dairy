# AI Diary Mobile (Flutter)

This folder will host the Flutter app (iOS + Android).

## Create the app scaffold

From the repo root:

```bash
flutter create mobile
```

Then wire the API base URL (e.g. `http://10.0.2.2:3000` on Android emulator) and implement:

- Onboarding (name, preferences, persona)
- Chat screen (text first)
- Voice record + STT (Whisper) next
